# Digital Forensics Case Study 01
## Disk Imaging with FTK Imager

### Objective
To acquire a forensic image of a removable storage device while maintaining evidence integrity.

### Tools Used
- FTK Imager
- Windows Operating System

### Procedure
1. Connected the flash drive.
2. Opened FTK Imager.
3. Selected "Create Disk Image".
4. Chose the physical drive.
5. Selected E01 image format.
6. Specified destination folder.
7. Generated MD5 and SHA1 hashes.
8. Completed image acquisition.

### Findings
- Successful acquisition of forensic image.
- Hash values generated for integrity verification.
- Image preserved original evidence without modification.

### Importance
Disk imaging ensures investigators work on a copy of evidence rather than the original device.

### Lessons Learned
- Always verify image integrity using hashes.
- Maintain chain of custody.
- Document every forensic action.
